Это премиум версия программы, пользуйтесь функцииями практически без ограничений!

Описание:
Эта программа имеет набор скриптов для личного использования, которые вы можете использовать без ограничений! 

Установка...
1. Установите Termux
2. Возьмите папку с программой, и переместите ее в папку Documents
3. Откройте Termux и скачайте python введя: (pkg install python)
4. Дайте доступ к каталогу файлов введя: (termux-setup-storage)
5. Перейдите в папку с программой введя: (cd storage/documents/X0ne)
6. Для запуска введите: (python Start.py)

Дополнительно:
7. Если программа просит установить модули, установите их введя: (pip install название модуля) Название написано в ошибке при запуске программы, после установки всех модулей, запустите программу заново...
